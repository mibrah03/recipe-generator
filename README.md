# 🎲 What Should I Cook?

An AI-powered recipe generator. Pick a cuisine, choose a meal type or enter ingredients you have at home, and get a random recipe instantly.

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Vercel auto-detects Vite — just click Deploy!

## Built With

- React + Vite
- Claude AI (recipe generation)
- Wikimedia Commons (food photos)
